# 💻 Meu Portfolio Pessoal

Bem-vindo ao repositório do meu portfolio pessoal! Este projeto é uma página web desenvolvida para apresentar meu portifolio.

A estrutura foi otimizada com foco em performance e organização, utilizando CSS estruturado, Font Awesome para ícones, scripts de interatividade e Tailwind CSS.

---

## 📂 Estrutura do Projeto

Com base na estrutura de pastas e arquivos exata do repositório, os arquivos estão organizados da seguinte forma:

```bash
├── css/
│   ├── all.min.css         # Estilos gerais minificados
│   └── style.css           # Folha de estilo principal da aplicação
├── icon/
│   ├── fa-brands-400.ttf   # Fontes do Font Awesome (Marcas)
│   ├── fa-brands-400.woff2
│   ├── fa-regular-400.ttf  # Fontes do Font Awesome (Estilo Regular)
│   ├── fa-regular-400.woff2
│   ├── fa-solid-900.ttf    # Fontes do Font Awesome (Estilo Sólido)
│   ├── fa-solid-900.woff2
│   ├── fa-v4compatibility.ttf
│   └── fa-v4compatibility.woff2
├── images/
│   ├── will.png            # Foto de perfil/exibição do portfólio
│   └── will_original_backup.png
├── js/
│   ├── main.js             # Script principal com funções dinâmicas
│   └── tailwind.js         # Configurações/integrações adicionais do Tailwind
├── index.html              # Ponto de entrada (HTML principal do portfólio)
└── README.md               # Documentação do projeto
```

---

## 🛠️ Tecnologias e Ferramentas

O desenvolvimento deste site faz uso das seguintes tecnologias:

*   **HTML5:** Estruturação semântica de todas as seções.
*   **CSS3 & Tailwind CSS:** Estilização moderna e utilitária para garantir um design totalmente responsivo e rápido.
*   **JavaScript (ES6+):** Manipulações de DOM e comportamentos interativos.
*   **Font Awesome:** Pacote de ícones vetoriais utilizados na interface.
*   **Git & GitHub Pages:** Versionamento de código e hospedagem nativa.

## 🚀 Como Executar Localmente

Para clonar e testar o portfólio na sua máquina:

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/williammenezesnascimento/williammenezesnascimento.github.io.git
   ```

2. **Navegar até a pasta do projeto:**
   ```bash
   cd williammenezesnascimento.github.io
   ```

3. **Abrir o arquivo index.html:**
   Basta abrir o `index.html` diretamente em seu navegador ou utilizar o **Live Server** no VS Code.
